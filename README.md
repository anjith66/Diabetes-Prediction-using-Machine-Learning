# 🩺 Diabetes Prediction using Machine Learning

A machine learning project that predicts whether a person is likely to have diabetes based on medical attributes. This project uses Python and Scikit-learn to build and evaluate a predictive model.

---

## 📌 Project Overview

Diabetes is a chronic disease that affects millions of people worldwide. Early prediction can help in better diagnosis and treatment.

This project uses a dataset of medical features such as glucose level, BMI, age, etc., to predict the likelihood of diabetes using machine learning algorithms.

---

## ⚙️ Technologies Used

- Python 🐍  
- NumPy  
- Pandas  
- Scikit-learn  
- Jupyter Notebook  

---

## 📊 Dataset Features

The model is trained on the following features:

- Pregnancies  
- Glucose  
- Blood Pressure  
- Skin Thickness  
- Insulin  
- BMI (Body Mass Index)  
- Diabetes Pedigree Function  
- Age  

---

## 🧠 Machine Learning Workflow

1. Data Collection  
2. Data Preprocessing  
3. Feature Scaling (StandardScaler)  
4. Model Training  
5. Model Evaluation  
6. Prediction
